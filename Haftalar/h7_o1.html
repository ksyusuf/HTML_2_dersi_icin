TEXT ÖZELLİKLERİ

Form ile gönderilen verilerin alınması için kullanılan PHP komutları Web Tarayıcı programında değil Web Sunucudaki PHP yorumlayıcı yazılımı tarafından çalıştırılabilir. Ek olarak php kodu bulunan bir web sayfasının uzantısı da “html” değil “php” olmalıdır. Bu sebeple veri alma işlemi için yazılan web sayfaları web sunucuya gönderilip oradan çalıştırılabilir.

Bilgisayarımızın yerel web sunucu olarak çalışması için üç yazılımın yüklü ve koordineli çalışması gerekir. Bunlar; web sunucu yazılımı, PHP yorumlayıcı yazılımı ve veri tabanı yönetim sistemi yazılımı. Bunlar bilgisayarımıza ayrı ayrı yüklenip ayarları yapılarak koordineli çalışması sağlanabilir. Fakat çok daha kolay bir yöntem daha vardır. Bu üç yazılımı içeren paket yazılımlar vardır. Bu paket yazılımı yüklediğimizde bilgisayarımız yerel web sunucusu gibi çalışmaya başlar.
